## How It Works

The agent connects to Gmail via OAuth and processes your inbox in batches:

1. **Classify** — labels each thread (action-needed, FYI, newsletter, etc.)
2. **Prioritise** — surfaces urgent threads first
3. **Draft** — generates reply suggestions using conversation context
4. **Confirm** — shows you a diff of proposed actions before anything is sent or archived

## Safety First

Inbox Zero is designed with a *human-in-the-loop* for all write operations. It reads autonomously but writes only after explicit approval. Destructive actions (archive, delete) are shown in a confirmation UI.

## Current Limitations

- Works with Gmail only (Outlook support planned)
- Reply drafts sometimes miss tone nuance for long-running threads
- Rate limits on Gmail API cause occasional delays for large inboxes
